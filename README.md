# 🚀 Ikman Web Scraper

A **powerful web scraper** for extracting listings from [Ikman.lk](https://ikman.lk) and saving the data in JSON format for analysis or automation.

---

## ✨ Features

- 🕵️ Scrapes listings data (title, price, location, etc.)  
- 💾 Stores data in JSON  
- 🔄 Handles multiple categories  
- 🛠 Easy to extend and customize  
- 📊 Ready for data analysis  

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/nishadii99/ikman-web-scraper.git

Navigate to project folder:

cd ikman-web-scraper

Install dependencies (Java + Maven):

mvn clean install
🏃‍♂️ Usage

Run the main Java class:

mvn exec:java -Dexec.mainClass="lk.ijse.Main"

✅ Data will be saved in data.json

🗂 Project Structure
Ikman Web Scraper/
├─ src/
│  └─ main/java/lk/ijse/Main.java
├─ data.json
├─ pom.xml
└─ .gitignore
💡 How It Works

Sends HTTP requests to Ikman.lk

Parses HTML using Java libraries

Extracts relevant listing details

Saves results in JSON format
